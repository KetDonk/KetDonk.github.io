JS-to-Brainfuck

A JavaScript to Brainfuck code translator

View on GitHub

Overview

I was bored one day and decided to make a JavaScript to Brainfuck code translator. I'm not sure why I did this, but I'm glad I did.

I now feel fulfilled in life. Most people can't say they've made a JavaScript to Brainfuck code translator, But i can.

Tech Stack

JavaScript

Key Features

Console.log Translation

Converts JavaScript console.log statements into Brainfuck code that produces the same output.

String Support

Handles string literals and converts them to Brainfuck character sequences.

Number Support

Converts numeric values into Brainfuck code that outputs the same numbers.

Optimized Output

Uses efficient Brainfuck patterns to minimize code length and execution time.

Example Usage

const translator = new JSToBrainfuck();
const jsCode = `
console.log("Hello");
console.log(42);
`;

const brainfuckCode = translator.convert(jsCode);
console.log('Generated Brainfuck code:');
console.log(brainfuckCode);
